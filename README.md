# circuit-forest

## Environment Setup

**Backend & Middle-End**

- Rust: https://rust-lang.org/tools/install/ (you may also install "rustup" from your package manager)
- `rust-analyzer`: https://marketplace.visualstudio.com/items?itemName=rust-lang.rust-analyzer
- `cargo doc`: very rudimentary docs

**Frontend**

- Node.js: https://nodejs.org/en/download (you may also install "node" from your package manager)
- `pnpm`: https://pnpm.io/installation (may be easiest to install via npm: `npm install -g pnpm`)

## Frontend Development

```sh
pnpm install
pnpm start:frontend
```
