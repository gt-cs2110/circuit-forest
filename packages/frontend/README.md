# CircuitSim Frontend

This is an Electron app using [Vue.js 3](https://vuejs.org/).

## Development

### Tools

Make sure you have [Node.js](https://nodejs.org/en/download) and [`pnpm`](https://pnpm.io/installation). If you're working with VS Code, I strongly recommend installing the [Vue](https://marketplace.visualstudio.com/items?itemName=Vue.volar), [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode), [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint), and [Tailwind CSS](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) extensions for proper intellisense.

### Developing

1. Clone this repository.
2. Run `pnpm install` to get the required dependencies.
3. Run `pnpm start` to start a local development server.
